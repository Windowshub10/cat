Replication :-

* First create a folder named data
* In the data folder create 3 more folders named rs0, rs1, rs2
* Then follow the below steps:

Step 1: Open Terminal
Enter cmd: mongod --port 27017 --dbpath "path of rs0 folder" --replSet "rso"
Run it

Step 2: Open another terminal
Enter cmd: mongod --port 27018 --dbpath "path of rs1 folder" --replSet "rso"
Run it

Step 3: Open again another terminal
Enter cmd: mongod --port 27019 --dbpath "path of rs2 folder" --replSet "rso"
Run it

Step 4: Again open another terminal
Enter cmd: mongo --port 27017

Write query: rs.initiate({
   _id: "rso",
   members: [
      { _id: 0, host: "localhost:27017" },
      { _id: 1, host: "localhost:27018" },
      { _id: 2, host: "localhost:27019" }
   ]
})

Then run: rs.add("localhost:27018")
          rs.add("localhost:27019")
          rs.isMaster()
          rs.status()
          exit

Step 5: Import existing database
Cmd: mongoimport --db Hole --collection Hole --file "C:\User\NQT\student.json" --jsonArray


Step 6: Again go to Mongo port 27017
Cmd: mongo --port 27017
Write query: show dbs
             use Hole
             show collections
             db.Hole.find()
             exit


Step 7: Then go to Mongo port 27018
Cmd: mongo --port 27018

Write query: rs.slaveOk()
	     show dbs
	     use Hole
	     show collections
	     db.Hole.find()
	     exit


Step 8: Then go to Mongo port 27019
Cmd: mongo --port 27019

Write query: rs.slaveOk()
	     show dbs
	     use Hole
	     show collections
	     db.Hole.find()
	     exit
